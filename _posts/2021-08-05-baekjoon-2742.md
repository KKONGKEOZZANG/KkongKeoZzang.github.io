# [2742번 : 기찍 N](https://www.acmicpc.net/problem/2742)

#### 1. 문제읽기
---

> N 찍기를 반대로 출력하는 문제   

`반복문`을 이용하면 쉽게 해결할 수 있다.  



#### 2. 제출 코드 
---



```python
n = int(input())

for i in range(n):
    print(n-i)
```
