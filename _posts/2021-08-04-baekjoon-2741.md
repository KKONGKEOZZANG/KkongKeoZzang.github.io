# [2741번 : N 찍기](https://www.acmicpc.net/problem/2741)

#### 1. 문제읽기
---

> 별찍기와 비슷한 문제   

`반복문`을 이용하면 쉽게 해결할 수 있다.  



#### 2. 제출 코드 
---

`while문` 사용하여 완료!  
`for문`으로도 가능하다.  


```python
n = int(input())

i = 1
while i <= n:
    print(i)
    i += 1
```
